# Multisim Circuit Collection

This repository contains various analog circuit simulations created using NI Multisim. Each folder includes a ready-to-run `.ms14` simulation file and represents a unique concept in analog electronics.

## ðŸ“‚ Circuit List

### 1. Astable Multivibrator
- **Folder**: `Astable_Multivibrator`
- **Description**: Generates a continuous square wave using two BJTs.
- **Applications**: Pulse generation, waveform generators.

### 2. Monostable Multivibrator
- **Folder**: `Monostable_Multivibrator`
- **Description**: 555 Timer-based monostable circuit that generates a single pulse when triggered.
- **Applications**: Timers, delay circuits.

### 3. Bandpass Filter
- **Folder**: `Bandpass_Filter`
- **Description**: Active bandpass filter using op-amp.
- **Applications**: Audio filtering, communication systems.

### 4. Band Rejection Filter
- **Folder**: `Band_Rejection_Filter`
- **Description**: Twin-T notch filter that blocks a specific frequency.
- **Applications**: Hum/noise removal.

### 5. Class A Amplifier
- **Folder**: `ClassA_Amplifier`
- **Description**: Basic single-transistor amplifier with resistive biasing.
- **Applications**: Audio amplification.

### 6. Voltage Series Feedback Amplifier
- **Folder**: `Voltage_Series_Feedback`
- **Description**: Feedback amplifier designed to improve gain stability.
- **Applications**: Signal amplification with controlled feedback.
