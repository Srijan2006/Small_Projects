# Eagle Circuits Collection

This repository contains schematic and board files for basic yet essential electronic circuits designed using Autodesk Eagle. Each project folder includes a fully functional circuit with documentation.

## Projects
1. **Wheatstone Bridge** – Precision resistor measurement setup using a balanced bridge configuration.
2. **Voltage Divider** – Simple voltage-level adjustment using resistor ratios.

Perfect for learners, experimenters, or anyone brushing up on analog fundamentals.
